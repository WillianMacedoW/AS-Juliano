# Mini E-commerce React

Aplicação de mini e-commerce usando React, React Router v6, Context API, TailwindCSS e JSON Server para simular a API.

O projeto foi criado com Vite e já inclui todas as dependências no `package.json` (React, React Router, Tailwind e JSON Server). Basta baixar o repositório, instalar as dependências e executar os comandos abaixo.

## Como rodar

1. Instale as dependências:
 ```bash
  npm install
  ```
2. Inicie o JSON Server em outro terminal:
   ```bash
   npm run serve:api
   ```
3. Rode o front-end:
   ```bash
   npm run dev
   ```

A API fica disponível em `http://localhost:3001/products`.
